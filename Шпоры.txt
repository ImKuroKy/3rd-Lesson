Шпаргалка по методам массива:

.join() - Изменяет стандартное разделение элементов массива на кастомное

.concat() - Дополняет элементы массива указанными элементами в параметрах

.reverse() - Отзеркаливает массив

.slice() - Заменяет массив на указанный отрезок

.splice(a, b, c) - "Вырезает" указанный отрезок из массива, где
1) a - Начальный индекс
2) b - Количество удаляемых элементов
3) c - Элементы для вставки вместо удаленных

.sort() - Сортирует массив по возрастанию (если элементы Int) или по алфавиту (если элементы string)
Также, может принимать функцию для кастомной сортировки

.push() - Добавляет элемент(-ы) в конец массива, возвращаемое значение - кол-во элементов массива

.pop() - Удаляет последний элемент массива

.shift() - Удаляет первый элемент массива, возвращает - первый элемент массива

.unshift() - Добавляет элемент(-ы) в начало массива
